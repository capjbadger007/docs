## LW003-B LoRaWAN Bluetooth Gateway

The LW003-B is a BLE-LoRaWAN gateway integrating LoRaWAN and Bluetooth wireless communication,
LW003-B can scan BLE Beacon data and transmit data to a LoRaWAN-Based gateway, and then upload to a
server, so as to realize environmental monitoring and indoor positioning.
