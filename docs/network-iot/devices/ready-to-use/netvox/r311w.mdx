## R311W Wireless 2-Gang Water Leak Detector

The R311W is a water leak sensor based on LoRaWAN protocol (Class A) carrying leak detection and
alarm function. When the R311W detects a leak, it will send an alarm message to the gateway. When
the sensor detects no leak, it will send a message that reports the status to the gateway. There are
two water leak sensors that allow users to monitor two spots.
