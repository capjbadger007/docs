## R311A Wireless Window Sensor

The R311A is equipped with a reed sensor, which can be used to detect the status of the door and
window. When the window is opened, R311A sends an alarm message to the control center. R311A
utilizes the latest ultra-low power consumption technology and requires no wiring. It requires just
button batteries to support its operation.
