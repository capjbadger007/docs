## R718AB Wireless Temperature and Humidity Sensor

The R718AB is a wireless communication device that detects ambient air temperature and humidity. The
R718AB can detect the temperature and humidity of the air and transmit the detected data to a data
center through the wireless network.

### Specifications

#### LoRaWAN device type:

- Class A end-device

#### Sensors:

- ambient temperature
- humidity
- battery

#### Dimensions:

\*\* Main device:

- width: 65 mm
- length: 112 mm
- height: 32 mm

\*\* Sensor cover:

- diameter: 16 mm
- length: 27 mm

#### Weight:

- 141 grams

#### Battery:

- type: 2x ER14505 Lithium AA battery / 3.6 V / 2400 mAh per battery
- replaceable: True

### Note:

- The base is attached to a magnet that can be attached to a ferromagnetic material object.
