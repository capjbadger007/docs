## R311G Wireless Light Sensor

The R311G built-in photosensitive sensor can be used for external ambient light intensity detection.
The R311G is the environmental illuminance automatic detector and the reporter which can send out
the environmental illumination report and is a low-power, long-range, wireless sensor that can
accurately detect environmental illuminance indoors.
